# Agent Rules

- Do not claim robot motion succeeded unless there is run-log evidence.
- Do not invent experiment results, metrics, photos, or videos.
- Do not commit secrets, credentials, private school data, or API keys.
- Large videos, ROS bags, and datasets should be referenced from docs/08-media-index.md, not committed directly.
- Keep raw evidence in docs/04-run-log/, docs/05-failure-log.md, and docs/06-metrics.md.
- Put publishable article material in docs/09-public-notes.md.
- Never run git reset --hard / git clean / git push --force; sync upstream with merge, not rebase, and not the GitHub "Sync fork" button.
